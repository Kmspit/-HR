⏱️ การเช็คอิน / เช็คเอาท์ ▼
  1. 1ไปที่เมนู ลงเวลางาน
  2. 2กด เช็คอิน เมื่อเริ่มงาน — ระบบถ่ายรูปและบันทึก GPS อัตโนมัติ
  3. 3กด พักเที่ยง เมื่อออกพัก และ กลับจากพัก เมื่อเริ่มงานต่อ
  4. 4กด เช็คเอาท์ เมื่อเลิกงาน
📸 การสแกนใบหน้า ▼
  1. 1ไปที่หน้า ลงเวลางาน และเลือกประเภทการลงเวลา
  2. 2กด สแกนใบหน้า — ระบบเปิดกล้องอัตโนมัติ
  3. 3หันหน้าตรงเข้ากล้อง ระบบจะตรวจจับและยืนยันตัวตนเอง
  4. 4รอข้อความยืนยัน เช็คอินสำเร็จ / ล้มเหลว
📅 การขอลาหยุด ▼
  1. 1ไปที่เมนู ขอหยุด
  2. 2เลือกประเภทการลา กำหนดวันที่ และใส่เหตุผล
  3. 3กด ส่งคำขอ — ระบบส่งแจ้งเตือน HR ทันที
  4. 4ติดตามสถานะในหน้าเดิม: รออนุมัติ / อนุมัติ / ปฏิเสธ
📍 การออกนอกสถานที่ ▼
  1. 1ไปที่เมนู ออกนอกสถานที่
  2. 2กด + เพิ่มแผนงาน
  3. 3กรอกวันที่ เวลา สถานที่ปฏิบัติงาน ชื่อเจ้าของกิจการ ประเภทงาน และระยะทาง
  4. 4กด บันทึก — ระบบส่งคำขอรอ CEO อนุมัติทันที
  5. 5แก้ไขได้หากสถานะยังเป็น รออนุมัติ โดยกดปุ่มแก้ไขที่แถวนั้น
🧾 การดูสลิปเงินเดือน ▼
  1. 1ไปที่เมนู สลิปเงินเดือน
  2. 2เลือกเดือนที่ต้องการดู
  3. 3กด ดาวน์โหลด PDF เพื่อบันทึกสลิป
🗓️ การดูปฏิทินและแผนงาน ▼
  1. 1ไปที่เมนู ปฏิทิน
  2. 2ดูวันหยุด งานนัดหมาย และกิจกรรมขององค์กร
  3. 3กดที่วันที่ในปฏิทินเพื่อดูรายละเอียดกิจกรรม
✅ การอนุมัติคำขอลา ▼
  1. 1ไปที่เมนู อนุมัติ
  2. 2ดูรายการคำขอที่รออนุมัติ
  3. 3กด อนุมัติ หรือ ปฏิเสธ พร้อมใส่หมายเหตุ (ถ้ามี)
  4. 4ระบบส่งแจ้งเตือนกลับไปยังพนักงานอัตโนมัติ
🔔 การออกใบเตือนพนักงาน ▼
  1. 1ไปที่เมนู แจ้งเตือน → กด สร้างใบเตือน
  2. 2เลือกพนักงาน ประเภทความผิด และระดับโทษ
  3. 3ระบุรายละเอียดและวันที่ → กด บันทึก
  4. 4ระบบบันทึกในประวัติและแจ้งเตือนพนักงานอัตโนมัติ
📍 การดูตำแหน่ง GPS พนักงาน ▼
  1. 1ไปที่เมนู ลงเวลางาน → กดแท็บ แผนที่ / GPS
  2. 2ดูตำแหน่งเช็คอิน/เช็คเอาท์ของพนักงานแต่ละคน
  3. 3กรองตามวันที่หรือสาขา
👥 การจัดการพนักงาน ▼
  1. 1ไปที่เมนู พนักงาน
  2. 2กด เพิ่มพนักงาน → ใส่ข้อมูลส่วนตัว บทบาท และแผนก
  3. 3แก้ไขข้อมูลได้โดยกดปุ่ม ⋮ ที่แถวพนักงาน
  4. 4กำหนดสิทธิ์เข้าถึงระบบได้จากแท็บ บทบาท
💰 การจัดการเงินเดือน ▼
  1. 1ไปที่เมนู เงินเดือน → เลือกรอบเงินเดือน
  2. 2ตรวจสอบชั่วโมงทำงาน OT และการขาด/ลา
  3. 3กด คำนวณ ตรวจสอบผลลัพธ์ แล้วกด ยืนยัน
  4. 4ระบบออกสลิปให้พนักงานแต่ละคนอัตโนมัติ
📈 การดูรายงานรายเดือน ▼
  1. 1ไปที่เมนู รายงานรายเดือน
  2. 2เลือกเดือนและแผนกที่ต้องการดู
  3. 3ดูสรุปการลา OT และชั่วโมงทำงานรวม
  4. 4กด ส่งออก Excel เพื่อดาวน์โหลด
